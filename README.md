# 🎬 CineIA

Sistema de recomendação de filmes que utiliza IA para sugerir títulos similares com base em gênero e sinopse.

## 🛠️ Tecnologias
* Python
* Streamlit (Interface)
* Pandas (Dados)
* Scikit-learn (IA / TF-IDF)
* Deep-translator (Tradução)

## 🚀 Como usar
1. Clone o repositório:
   ```bash
   git clone https://github.com/Helsiva/CineIA-Sreamlit.git
2. Instale as bibliotecas necessárias:
   ```bash
   pip install -r requirements.txt
3. Rode o projeto:
   ```bash
   streamlit run recomendacao_filme.py
